# Postman Collections

Thư mục này chứa collection kiểm thử cho OpenAPI contract AI Vision / Access / Core Policy. Sử dụng collection `FIT4110_lab05_iot_compose.postman_collection.json` cùng environment `FIT4110_lab05_local.postman_environment.json` để chạy Newman trên stack Compose.

Đảm bảo `baseUrl` được đặt thành `http://localhost:8000` và `authToken` được giữ là `local-dev-token` cho môi trường local.
